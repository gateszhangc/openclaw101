# OpenClaw 从零陪跑

这是一套面向普通用户的 OpenClaw 中文教程草稿，写法参考 `openclaw101.dev/day` 的节奏，但内容以 2026 年 3 月的官方文档为准。

你会在正文里看到一个陪跑角色：**虾滑**。

- `OpenClaw` 是平台名
- `虾滑` 是教程里的原创助手角色
- 教程的目标不是把文档翻译一遍，而是把“第一次上手最容易卡住的地方”讲明白

## 这套教程的主线

为了让新手先跑通再扩展，这套教程只保留一条默认主线：

1. 先在 Dashboard 里完成第一次对话
2. 再弄清楚模型和认证
3. 然后接上 Telegram
4. 接着写人格和长期记忆
5. 最后再装 Skills、开 heartbeat、加 cron
